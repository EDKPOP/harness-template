# Role Matrix

## Core roles
- Initializer: prepare or migrate harness state
- Planner: create approved implementation plan
- Implementer: change code and tests
- Reviewer: judge plan compliance and quality

## Conditional roles
- Code Explorer: search existing code and reuse opportunities
- Code Architect: shape larger technical changes
- Silent Failure Hunter: find fake green outcomes
- PR Test Analyzer: judge test quality
- Harness Optimizer: turn repeated failures into stronger harness rules
- Loop Operator: control retries, pauses, and stop-loss
- Council: advise on ambiguous structural decisions
